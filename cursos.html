Entidade Cursos (Courses)

Esta entidade irá representar os cursos cadastrados no sistema.

Operações HTTP

A URL para acessar o recurso (entidade) course é: https://hackjoy-api.herokuapp.com/courses

Novo curso


Método: POST
URL: https://hackjoy-api.herokuapp.com/courses/new

Para se cadastrar um novo curso é necessário informar os seguintes dados:

  • id_certificate: Id do certificado que o curso utilizará.
  • name: Nome do certificado.
  • about: Sobre o curso.
  • description: Descrição do curso.
  • image: Imagem do curso.

Exemplo:

Listar

Método: GET URL: https://hackjoy-api.herokuapp.com/courses

Retorna um JSON com todos os cursos cadastrados

Encontrar curso por id

Método: GET URL: https://hackjoy-api.herokuapp.com/courses/{id}

Retorna o cursos que possui o id informado por parâmetro.

Atualizar curso por id

Método: PUT URL: https://hackjoy-api.herokuapp.com/courses/{id}

Para se atualizar um curso é necessário informar os seguintes dados:

  • id_certificate: Id do certificado que o curso utilizará.
  • name: Nome do certificado.
  • about: Sobre o curso.
  • description: Descrição do curso.
  • image: Imagem do curso.
