Entidade Vídeo (Video)

Esta entidade irá representar os vídeos dos cursos.

Um curso possui N conteúdos, os conteúdos possui tipos, e a entidade Vídeo é um tipo de conteúdo.

Operações HTTP

A URL para acessar o recurso (entidade) video é: https://hackjoy-api.herokuapp.com/videos/new

Novo vídeo


Método: POST
URL: https://hackjoy-api.herokuapp.com/courses/new

Para se cadastrar um novo vídeo é necessário informar os dados para a entidade content e os dados para a entidade video:

  • content
    • id_course: Id do curso no qual se está cadastrando os conteúdos.
    • name: Nome do conteúdo.
    • sequence_number: Ordem na qual o conteúdo é mostrado na tela.
    • entity: Tipo de conteúdo, no caso deve ser informado o tipo "Video".
  • video
    • name: Nome do vídeo.
    • link: Link do vídeo.
    • workload: Carga horária do vídeo.

Atualizar vídeo por id

Método: PUT URL: https://hackjoy-api.herokuapp.com/videos/{id}

Para se atualizar um vídeo é necessário informar os seguintes dados:

  • content
    • name: Nome do conteúdo.
    • sequence_number: Ordem na qual o conteúdo é mostrado na tela.
    • entity: Tipo de conteúdo, no caso deve ser informado o tipo "Video".
  • video
    • id: Id do vídeo.
    • name: Nome do vídeo.
    • link: Link do vídeo.
    • workload: Carga horária do vídeo.
